CSS Exemplo de Alinhamento

Setembro de 2014

A propriedade text-align faz exatamento o que o nome sugere: alinha o texto da mesma forma que ocorre com os editores de texto. Existe quatro tipos de alinhamento para texto em HTML com CSS:

  • centralizado (center),
  • justificado (justify),
  • esquerda (left) e
  • direita (right).
